This CDRom contains a selection of modules designed for Dolibarr ERP & CRM, the opensource software for business.
This software can be downloaded, used, modified and distributed for free on page: http://www.dolibarr.org

They are all available for the official market place for a fee at:
http://www.dolistore.com

Modules are provided as is with no support but you may find help on dolibarr forums:
http://www.dolibarr.org/forum


Installation of a module:
1) Unzip the selected module found into this cdrom into this dir related to your Dolibarr version. You should get after that, on your server, a new directory yourdolibarrrootdir/htdocs/newmodule
2) Go into Dolibarr, menu Home - Setup - Modules. You should find new module into one of the tab list to enable it.

License/author of modules:

Installation d'un module:
1) Dézipper le module trouvé sur ce cdrom dans le répertoire correspondant à votre version de Dolibarr. Vous devrier avoir, après cela, sur votre serveur, un nouveau répertoire du type racinedolibarr/htdocs/nouveaumodule
2) Aller dans Dolibarr, menu Accueil - Configuration - modules. Vous devriez trouver le nouveau module dans un des onglets listes pour l'activer.

Licence/Editeur des modules:
Pour connaitre la license et l'éditeur d'un module, aller sur http://www.dolistore.com pour retrouver le module et regardez sa description.
